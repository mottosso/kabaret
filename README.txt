=======
Kabaret
=======

Kabaret is a high-level python digital creation framework providing 
robustness and scalability to studios of any size.

Kabaret is funded by the `Supamonks Studio <http://www.supamonks.com>`_ and 
is currently in design stage.

Documentation
=============
Kabaret is made of several packages. 
The documentation covers all of them and is here: http://packages.python.org/kabaret
